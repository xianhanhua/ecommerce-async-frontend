# 电商前端异步请求项目

本项目用于完成“为电商前端添加异步请求功能”的实验任务。项目使用原生 HTML、CSS、JavaScript 实现，不依赖后端服务，通过 `request.js` 封装 Mock API 请求，每次分页查询和分类筛选都会触发一次异步请求。

## 运行方式

直接双击：

```text
打开项目.bat
```

或打开：

```text
src/index.html
```

## 功能

- 独立请求工具：`src/js/request.js`
- Mock 商品接口：`src/js/mock-api.js`
- 商品分页查询
- 商品分类筛选
- 关键字查询
- 每次点击上一页、下一页、分类切换、搜索都会发送一次异步请求
- 页面展示请求日志，便于截图作为接口测试证明
